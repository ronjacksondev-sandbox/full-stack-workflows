# Write HTML Boilerplate

## Create file
```bash
touch index.html
```

## Open VS Code
```bash
code .
```

## Write `html` and `head` elements
- Start with `<!DOCTYPE html>`
- Set `html lang="en"`

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Duo Budget – A Clear View of Your Time and Money</title>
    <meta name="description" content="Duo Budget helps you understand how your time and money work together using shared categories.">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  </head>
  <body>
  </body>
</html>
```
