# Write HTML Body

## Add the page structure inside the `body`
- Add a `header`
- Add a `main` placeholder
- Add a `footer`

```html
<body>
  <header>
    <nav>
      <div>
        <label>Adjust Font Size:</label>
        <div>
          <button id="setSmallFontSizeButton">Small</button>
          <button id="setMediumFontSizeButton">Medium</button>
          <button id="setLargeFontSizeButton">Large</button>
        </div>
      </div>
    </nav>
    <h1>Duo Budget</h1>
    <p>A clear view of your time and money in one place.</p>
  </header>

  <main>
    <!-- Main content will be added next -->
  </main>

  <footer>
    <p>Duo Budget is an early-stage project. All content subject to change.</p>
    <p>&copy; Duo Budget</p>
  </footer>
</body>
```
