# Build

### Run vite Build
- build produced in /dist folder
- bundles js into file(s) in assets folder
    - Your application code
    - React
    - React DOM
    - Vite runtime helpers
- compiles JSX to JavaScript
- minifies code
- removes dead code (tree shaking)
- dependencies are bundled
- index.html generated with link to bundled js file

``` bash
npm run build
```
