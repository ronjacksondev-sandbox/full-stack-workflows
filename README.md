# Full Stack Workflows

These workflows are sets of small tutorial projects that you can build and deploy to develop full-stack muscle memory.

## How do I use this?
This is an informational content repository and is intended to be read on GitHub pages at 
[Full-stack Workflows](https://ronjacksondev-sandbox.github.io/full-stack-workflows/).  

## Run locally
If you want to view the site offline/locally, you can clone the repo and run the following command.
``` bash
npm run serve
```

## Tech Stack
This site is a markdown SSG site built with 11ty.
