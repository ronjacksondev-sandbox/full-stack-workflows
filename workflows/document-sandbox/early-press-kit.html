Product summary

Duo Budget is a simple tool that helps people understand how their time and money work together.

Key features (MVP scope)

Feature Description
Shared categories One set of categories for both time and money.
Log time entries Record hours tied to shared categories.
Log money entries Track dollars using the same structure.
Category totals Summaries showing where resources go.
Unified dashboard Basic view of total hours and dollars.
User login Secure access to saved data.

Problem and solution

Problem

Most people track time and money in separate apps, spreadsheets, or not at all. This makes it difficult to understand how both resources relate.

Solution

Duo Budget connects time and money using a single category system.

Founder statement

Time and money are always connected, yet the tools we use treat them as separate worlds. Duo Budget brings them together.

Audience

  • Busy people who want clarity without complexity
  • Planners who like seeing the big picture
  • Curious users who want to understand their habits

Boilerplate
