# Setup Folders

### Setup root folders
``` bash
# source code
mkdir src
# SQL migrations or ORM migrations
mkdir migrations 
# Seed data
mkdir seeds
# Code tests
mkdir tests
```

### Setup sub folders
``` bash
# DB infrastructure
mkdir db
# DB queries/data access
mkdir models
# Validation logic
mkdir validation
# Cross cutting concerns
mkdir middleware
# Business logic
mkdir services
# Request pipeline
mkdir controllers
# Request config
mkdir routes
# Static files
mkdir public
mkdir public/css
mkdir public/js
mkdir public/images
# View pages
mkdir views
mkdir views/layouts
mkdir views/partials

# Logging etc.
mkdir utils

```
